# 🌦️ Weather App

A simple and responsive weather application that allows users to search for the current weather conditions of any city in the world. The app fetches real-time weather data using the OpenWeather API and displays temperature, weather description, and weather icons.

## ⚙️ How It Works

1. The user enters a city name.
2. JavaScript sends a request to the OpenWeather API.
3. The API returns weather data in JSON format.
4. The application extracts:

   * City name
   * Temperature
   * Weather description
   * Weather icon
5. The information is displayed dynamically on the page.

---

## 🔑 API Setup

This project uses the OpenWeather API.

1. Create an account on OpenWeather.
2. Generate your API key.
3. Replace the placeholder in `script.js`:

```
const apiKey = "YOUR_API_KEY";
```
